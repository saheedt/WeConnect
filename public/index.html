    Introduction

    Welcome to the WeConnect API! You can use our API to access WeConnect API endpoints, which can get information on various businesses and their reviews in our database.

    Users

    Sign up

    Request:

    
{
  "email": "user@address.com",
  "password": "somepassword"
}

    Response:

    {
  "message": "user registered successfully",
  "user": {
    "id": 1,
      "email": "user@address.com"
  }
}

    Request

    • Endpoint: POST: /api/v1/auth/signup

    Response

    • status: 201 Created
    • Body (application/json)

    Login

    Request:

    {
  "email": "user@address.com",
  "password": "somepassword"
}

    Response:

    {
  "message": "login success",

    Request

    • Endpoint: POST: /api/v1/auth/login

    Response

    • status: 200 Created
    • Body (application/json)

    location location to query If supplied, the result contains only businesses from the location.
    category category to query If supplied, the result contains only businesses from the category.

    Errors

    WeConnect API uses the following error codes:

    Error Code Meaning
    200 Ok -- Successful Request
    201 Created -- Post Request Successful
    400 Bad Request -- Your request is faulty
    401 Unauthorized -- Authentication error
    404 Not Found -- The specified user / business could not be found
